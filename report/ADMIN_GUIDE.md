# Sovereign Administration Guide v9.2

This guide outlines the operational protocols for managing the Sovereign Platform.

## 1. Accessing the Command Center
Navigate to `/admin.html` and authenticate via the **Executive SSO** portal using your authorized Firebase credentials.

## 2. Tab Navigation
- **Dashboard**: High-level telemetry (Users, Revenue, Delta).
- **Content**: Manage educational modules and 12-grade curriculum assets.
- **Store_Mgr**: Control digital listings in the Sovereign Store.
- **API_Matrix**: Real-time management of OpenRouter and Razorpay credentials.
- **Monitoring**: Live performance tracking and Cloud-Error logs via Firestore.

## 3. Session Security
Always use the **Terminate_Session** command when leaving the Command Center to ensure absolute vault integrity.
