## 🚀 Deployment Instructions

### 1. High-Performance Hosting (Netlify / Vercel / Hostinger)
1. **Upload**: Drag and drop the contents of the `TradeSovereign` folder into your hosting provider.
2. **Environment Variables**: Add your API keys (Razorpay, Supabase, etc.) in the dashboard.
3. **Site is Live**: Your platform will be accessible immediately via the production URL.

### 2. Custom Domain Setup
- Go to your Domain Settings.
- Add `tradesovereign.com`.
- Point your A/CNAME records to the server IP / subdomain provided by your host.
