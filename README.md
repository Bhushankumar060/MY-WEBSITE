# Sovereign v9.0 - Executive Build

## Deployment Instructions
1. **GitHub**: Push this entire directory to the main branch of your repository.
2. **Netlify**: Connect the repository to Netlify.
3. **Env Vars**: Inject `SUPABASE_URL`, `SUPABASE_ANON_KEY`, `RAZORPAY_KEY_ID`, and `OPENROUTER_API_KEY` into the Netlify Dashboard.
4. **Live URL**: Once deployed, the site will be live at your designated URL.

## Folder Structure
- `/admin/`: SSO Login Portal.
- `/js/`: Modularized Service Layer.
- `/report/`: Enterprise-grade documentation and security audits.
- `index.html`: Immersive 3D Landing Page.

## Access
- **Local Dev**: `python -m http.server 3000`.
- **Admin Root**: Use `/admin/login.html` to enter the Command Center.
